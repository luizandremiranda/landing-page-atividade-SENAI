# Portfólio Profissional - Lucas Almeida Ferreira

## 📌 Sobre o projeto

Este projeto foi desenvolvido como atividade do curso do **SENAI**, com o objetivo de criar uma página de apresentação profissional utilizando tecnologias de desenvolvimento web.

A proposta foi desenvolver um portfólio online para apresentar minha trajetória profissional, conhecimentos técnicos, projetos e objetivos na área de Tecnologia, com foco em **Desenvolvimento Web e Análise de Dados**.

Durante o desenvolvimento, utilizei o **ChatGPT como ferramenta de apoio** para auxiliar na organização das ideias, estruturação do código e melhorias na apresentação do projeto.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- Git e GitHub

---

## 💻 Funcionalidades

O projeto apresenta:

✅ Página inicial com apresentação profissional  
✅ Seção "Sobre mim"  
✅ Histórico profissional  
✅ Lista de tecnologias e habilidades  
✅ Área de projetos realizados  
✅ Competências técnicas e comportamentais  
✅ Seção de contato  

---

## 🎯 Objetivo

Criar um portfólio profissional para demonstrar conhecimentos em:

- Desenvolvimento Web
- Estruturação de páginas HTML
- Estilização com CSS
- Organização de projetos utilizando GitHub
